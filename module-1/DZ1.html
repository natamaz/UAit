• Создать новый проект с названием traveland. Проект должен содержать папки img, css, fonts, js и файил index.html в корне проекта



  • По ссылке скачайте картинки и перенесите их в свою папку.



  • По той же ссылке проинспектируйте шрифты, скачайте их гугла и поместите в вашу папку шрифтов



  • Структура нашей страницы: - div с классом wrapper - его дочерними элементами будут banner, section(Popular Location), section(Special Offers), section(From Blog & News) и footer - у каждой секции будет дочерний элемент div с классом container



  • Banner в контейнере содержит два дочерних элемента header и div с любым классом в котором будет лежать заголовок, текст и кнопка



  • В header будет логотип, менюшка и кнопка



  • Фон в секции баннер сделан с помощью background-image: url("путь к картинке");



  • Popular Location состоит из секции, контейнера. В контейнер помещаем два потомка, в первом будет лежать тайтл и текст, во втором будут три одинаковых блока



  • Логотип < a href="#" class="logo">< img src="../img/">< /a>



  • Меню < ul>< li>< a href="#">Home< /a>< /li>< /ul>



  • Кнопка < a href="#" class="btn">Кнопка< /a>



  • Заглавие в баннере < h1>, текст < p>



  • Заглавие в секциях < h2>, текст < p>

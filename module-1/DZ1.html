• Создайте новый проект который будет содержать index.html папку с картинками(добавьте в нее любую картинку), папку со стилями и фаил стилей(подключите его), папку со шрифтами
  • Страница должна быть на любом языке( leng изменить на язык страницы)
  • Создайте на странице div с классом wrapper который станет родителем для всех последующе созданных тегов
  • Создайте на странице header с классом header поместите в него div container
  • header должен содержать список ul li с вашим телефоном и адресом, список будет потомком container
  • создайте секцию с классом photo и контейнером в нем с помощью тега img подключите картинку
  • создайте секцию с классом name и контейнер
  • секция name должна содержать заголовок первого уровня с вашим именем, заголовок второго уровня с вашей фамилией
  • создайте секцию с классом info, дочерним элементом создайте div с классом container, создайте три сестринских div с классом box их родителем будет div с классом box-container, который в сво. очередь будет потомком container
  • создайте footer с классом footer и контейнеро в нем, в котором будет ссылка на любой ресурс с текстом "это ссылка"

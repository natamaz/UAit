Структура HTML-документа:

  1. "< !DOCTYPE html >"

    Он не относится к тегам и никаким образом не может отображаться на странице. Его задача — указать браузеру, какой стандарт HTML используется в этом документе.

  2. < html>< /html >

    Благодаря этому тегу браузер понимает, где начинается контент, который необходимо обработать как HTML.

  3. < html lang="en">

    В нем указывается язык, на котором отображается веб-страница

  4. < head>< /head >

    служит для хранения служебной информации. Такая информация называется метаинформацией. В современном вебе она отвечает не только за служебную информацию для браузера, но и активно используется при продвижении сайта. Поисковые системы считывают всю эту информацию и на основе множества алгоритмов определяют место сайта при разных поисковых запросах.

  5. < meta charset="UTF-8">

    Он позволяет установить кодировку документа. Универсальной кодировкой, которая содержит большинство необходимых символов из разных языков, является кодировка UTF-8.

  6. < title>Моя первая страница< /title>

    Для указания заголовка страницы используется специальный парный тег title, внутри которого указывается нужная информация.

  7. < body>< /body>

    является «телом» всей страницы, здесь размещается вся информация, которая будет выведена на странице.
